# Data Directory

This directory contains the raw audio datasets used in this project.

## Expected Structure

```
Data/
├── Macaque/                    # Macaque vocalization dataset
│   ├── AL/                     # Individual AL's vocalizations
│   ├── BE/                     # Individual BE's vocalizations
│   ├── IO/                     # Individual IO's vocalizations
│   ├── MU/                     # Individual MU's vocalizations
│   ├── QU/                     # Individual QU's vocalizations
│   ├── SN/                     # Individual SN's vocalizations
│   ├── TH/                     # Individual TH's vocalizations
│   └── TW/                     # Individual TW's vocalizations
│
├── Zebra finch/                # Zebra Finch vocalization dataset
│   ├── AdultVocalizations/     # Adult vocalizations
│   └── ChickVocalizations/     # Chick vocalizations
│
└── processed/                  # Generated by scripts (gitignored)
    ├── subsets/                # Created by script 02
    └── preprocessed_subsets/   # Created by script 02
```

## Dataset Information

### Macaque Vocalizations
- **Individuals:** 8 (AL, BE, IO, MU, QU, SN, TH, TW)
- **Total samples:** 7,285 vocalizations
- **Format:** Audio files (.wav or similar)
- **Task:** Individual identification

### Zebra Finch Vocalizations
- **Individuals:** 10 (subset used in experiments)
- **Total samples:** 3,433 vocalizations
- **Categories:** Adult and Chick vocalizations
- **Format:** Audio files (.wav or similar)
- **Task:** Individual identification and call type classification

## Data Not Included

**Note:** Raw data files are NOT included in this repository due to size constraints.

To run the experiments, you need to:
1. Obtain the Macaque and Zebra Finch datasets
2. Place them in the appropriate directories following the structure above
3. Run the preprocessing pipeline

## Processed Data

The `processed/` directory is automatically created when you run:
```bash
python scripts/02_create_subsets_and_preprocess.py
```

This generates:
- `subsets/` - Random subsets for quick experiments
- `preprocessed_subsets/` - Audio preprocessed to 16kHz, mono, normalized

Both subdirectories are gitignored as they can be regenerated from raw data.
